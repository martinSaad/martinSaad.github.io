- Modern, clean design
- Full mobile support (Responsive Design)

- Full accessibility

- Built-in contact form

## Technologies

- HTML5
- CSS3
- JavaScript (Vanilla)
- Formspree for forms
- Google Analytics
- AWS Amplify for hosting and deployment

## Local Installation

1. Clone the repository:
```bash
git clone https://github.com/username/personal-website.git
cd personal-website
```

2. Open the files in your preferred code editor

3. Run a local server (e.g., with Python):
```bash
python -m http.server 8000
```

4. Open your browser at `http://localhost:8000`

## Deployment

The website updates automatically with every push to the main branch via AWS Amplify.

### AWS Amplify Setup:

1. Create an AWS account
2. Set up a new Amplify app
3. Connect to GitHub repository
4. Configure custom domain

## Maintenance

- Update personal details in `index.html`
- Replace images in the `/images` directory
- Update Google Analytics code
- Update Formspree identifier
